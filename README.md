# 🚀 AI Resume Builder

An advanced **AI-powered Resume Builder** built using Streamlit and Llama 4.

---

## ✨ Features

* 🧠 AI Resume Generation
* 📊 ATS Score Analyzer
* 🎯 Job Description Tailoring
* 📝 Cover Letter Generator
* ❓ Interview Questions & Answers
* 🔍 Skill Gap Analysis
* 💼 LinkedIn Summary Generator
* 📄 PDF Export

---

## 🛠️ Tech Stack

* Python
* Streamlit
* SambaNova (Llama 4 API)
* SQLite
* ReportLab
